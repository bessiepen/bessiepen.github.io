Cheatsheet：完整的html文件结构

发表于 2016-08-24   |   分类于 技术笔记   |  

一个简单的完整html document包含了以下几个部分：

  1. doctype declaration: <!doctype html>
  2. root element: <html></html>
  3. head element: <head></head>
  4. body element: <body></body>
  5. nested element: <body><p>I'm a paragraph</p></body>
  6. attributes: in the start tag, eg: <html lang="zh">
  7. comments: <!-- this is comment text -->
