List of forms elements

element attributes info
<form> action, method, enctype forms
<textarea> name, form, placeholder, required, autofocus, disable, readonly text input area
<input> type, name, form, placeholder, required, autofocus, disable, readonly input control
<button> type, name, form, required, disable button
<label> for, form a caption for a form control
<select> name, form, required, autofocus, disable, multiple a control for selecting among a list of options
<option> label, value, disable, selected an option in a select control
<fieldset> name, form, disable grouping things
<legend> global attribute giving a title

Attributes

method

methodfeature
get
  1. 数据提交后显示在url上（不安全）；
  2. 字符有限制，不能提交大文件。
post
  1. 数据提交后不显示在url上（安全）；
  2. 可提交大文件，如images，video，file等；
  3. <form>必须有enctypeattribute。

▶︎ type=”text”

- Text is the default value of type attribute.
- Value attribute is optional.

▶︎ type=”radio/checkbox”
non-boolean attribute boolean attribute
⌦ placeholder checked: option is selected by default.
⌦ readonly

note:
- Value attribute is optional.

▶︎ type=”button”
non-boolean attribute boolean attribute
⌦ placeholder ⌦ required
⌦ readonly

note:
- It represents a button with no additional semantics.
- It has been replaced by <button> element in HTML5.
- Value attribute must be non-empty.

▶︎ type=”password”

note:
- For the sake of security, when use password-input element, <form> element must have attribute method="post".
